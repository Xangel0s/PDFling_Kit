# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.0] - 2026-03-06

### Added

- Popup UI refresh with animations and fixed action dock.
- Language toggle support (ES/EN) with persistence.
- Theme selector and dark mode sync between popup and workspace.
- Workspace shape tools (rectangle, ellipse, triangle) with color/size controls.
- Rounded rectangle toggle and figure popover UX.
- Undo/redo controls with keyboard shortcuts.
- PDF flatten export support for shapes and rounded corners.
- Help modal with project and repository support info.
- Branded icon set (16/24/32/48/128) wired to manifest.
- Local history persistence for export/compress output.

### Changed

- Renamed product references from Mini Sterling to PDFling in documentation.
- Updated README structure for enterprise-ready repository documentation.

### Removed

- Legacy design artifact files from `diseno/` working drafts.
